Command-line options

Usage: <ielauncher> [<option>:[argument]]

Examples:

  • <ielauncher> /URL:about:blank
  • <ielauncher> /URL:about:blank

Available options:

Switch Argument(s) Meaning
/url <url> Set the startup url for this session
/startupurl <url> Set the default startup url (for ALL sessions)
/gui, /GUI false / true or 0 / 1 If true or 1, show the GUI; If false or 0 (or anything else) this switch is ignored
