IELauncher's GUI (Graphical User Interface)

The IELauncher GUI looks like this:

The Startup URL textbox represents the URL opened by IELauncher on startup.

Clicking on the Run IE button will:

  • save the specified Startup URL (specified in the Startup URL textbox) and
  • start Internet Explorer, closing the IELauncher GUI.

Clicking on the Cancel button will close the GUI, taking no further action.

After changing the text of the Startup URL textbox, the Save button will become active. Clicking on it will save the specified Startup URL (specified in the Startup URL textbox).
